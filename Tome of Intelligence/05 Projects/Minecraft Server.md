## Vital Info
### Starting the server:

Open CMD prompt and type the following two commands:
->
**First**: cd C:\Source\Minecraft Server
->
**Second**: java -jar server.jar
### Joining the sever
For connecting to the server to **play Minecraft on the PC running the Minecraft server**, go to **Multiplayer** -> **Direct** **Connection** -> type **localhost** in the Server Address box and click join server.
